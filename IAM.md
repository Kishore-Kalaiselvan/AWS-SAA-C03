### What is IAM ?
- IAM - Identity and Management is a AWS service Used for Authentication & Authorization
- Root account creaed by default
- Users are people within your organisation and an be grouped
- Groups only contains users, not other groups
- Users dont have to belong to a group and user can belong to multiple groups

### What is IAM Policy
- Users or groups can be assigned JSON documents called policies
- These policies define the permissions of the Users
- In AWS you apply least privilege principle.
